# 📅 大格子互動月曆排程器 (Interactive Calendar Scheduler)

一個專為家庭、學生補習與多堂課排程設計的**超高密度 A4 單頁列印月曆系統**。基於 FullCalendar 6 開發，具備一鍵自動批次生成、意外週靈活單日調整，以及強大無視瀏覽器縮放的高解析度 PDF 導出功能。

👉 **點此線上使用：** [https://0xseanlee.github.io/calendar/](https://0xseanlee.github.io/calendar/)

---

## ✨ 獨家核心功能

* 🔒 **終極併行不換行壓縮：** 時間與內容完美壓縮至同一行（例如：`14:00數-S`、`18:30琴-v`），最大化利用大格子空間，即便排入再多行程也絕不切字。
* 📥 **高解析度 A4 PDF 導出：** 內建「背景虛擬獨立大畫布技術」，徹底無視使用者瀏覽器的縮放比例。點擊下載，保證 100% 輸出單頁完美、不跑位、不歪斜的橫式 A4 PDF 桌曆。
* 🔁 **每週固定批次排課（一鍵生成）：** 獨立的批次設定面板，可自由**複選星期幾**（例如勾選「星期一、三」），一鍵自動為未來 52 週（整份全年度）排滿固定課表，效率翻倍。
* ⚡ **意外週靈活無感調整：** 批次生成的行程在底層皆為獨立事件。遇到請假、調課等突發意外時，**點擊單日行程可直接進行「免防呆警告」快速秒刪或修改**，其他 51 週的同系列行程完好如初、絕對不被弄亂！
* 🌐 **全天行程置頂：** 支援整天活動（如：連假、全家出遊），自動隱藏時間碼並智慧置頂在格子最上方。
* 🔎 **動態字體大小槓桿：** 頂部內建字體大小調整滑桿（10px - 20px），網頁與下載的 PDF 同步縮放，自由掌控列印視覺。
* 防手殘洗腦防呆： 針對「刪除一整年全系列」與「清空所有紀錄」這類毀滅性操作，內建疯狂強調『取消』的終極防呆確認框。

## 👥 行程對象精簡對照

為了防範傳統行事曆因人名太長導致省略號 `...` 出現，系統全面採用極限精簡代碼：
* **S**：👦 Sean (質感藍)
* **v**：👧 Vivi (浪漫粉)
* **S+v**：👫 S+v (氣質紫)
* **全家**：👪 全家 (氣質紫)
* **其他**：✍️ 自訂備註

## 🛠️ 技術棧與外掛

* **前端核心：** HTML5, CSS3 (CSS Variables 變數動態控制), JavaScript (Vanilla JS)
* **日曆引擎：** [FullCalendar v6](https://fullcalendar.io/)
* **截圖引擎：** [html2canvas v1.4.1](https://html2canvas.hertzen.com/) (雙倍高解析渲染)
* **PDF 生成：** [jsPDF v2.5.1](https://github.com/parallax/jsPDF) (適應 A4 Landscape 比例)
* **資料持久化：** 瀏覽器 `localStorage` (自動儲存所有行程與粉紅假日設定，關閉網頁不遺失)

---

## 📝 使用與安裝說明

1. 將本專案 Fork 或 Clone 到你的電腦。
2. 確保目錄最外層有 `index.html`。
3. 前往專案的 **Settings -> Pages**。
4. 在 **Build and deployment** 下的 Source 選擇 `Deploy from a branch`。
5. Branch 選擇 `main` (或 `master`) 根目錄 `/ (root)`，點擊 **Save**。
6. 稍等 1 分鐘，即可透過你的專屬 GitHub Pages 連結開始排程與準備列印！
